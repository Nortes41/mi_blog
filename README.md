## Descripción de la práctica
En esta práctica he realizado el despliegue de una web estática utilizando control de versiones con Git y el servicio de alojamiento GitHub Pages.

## Pasos realizados
1.  **Preparación local:** Creación de la estructura HTML, CSS y JS.
2.  **Control de versiones:** Inicialización del repositorio y configuración de usuario.
3.  **Conexión remota:** Vinculación del repositorio local con GitHub vía HTTPS.
4.  **Commits atómicos:** Realización de cambios progresivos documentados paso a paso.
5.  **Despliegue:** Configuración de la rama `main` en GitHub Pages.
